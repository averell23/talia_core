base_for(url)

Will try to figure out the "base" (that is the parent directory or path) If the base is a directory, this will return the directory name, but if it is an URL, this will return an URI object.

open_from_url(url, options = {}) {|io| ...}

Opens the given (web) URL, using URL encoding and necessary substitutions. The user must pass a block which will receive the io object from the url.

The options may contain the http authentication information and such. See the documentation for open-uri for more information. Example for options: 

  :http_basic_authentication => [login, password]

open_generic(url, options = {}) {|io| ...}

Generic "open" method for files and urls. This won‘t choke on file:// URLs and will do some extra escaping on the URL.

See open_from_url for an explanation of the options
