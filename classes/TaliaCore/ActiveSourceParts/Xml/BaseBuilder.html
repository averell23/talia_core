Class TaliaCore::ActiveSourceParts::Xml::BaseBuilder
In: lib/talia_core/active_source_parts/xml/base_builder.rb
Parent: Object

Base class for builders that create source-related XML. This uses a Builder::XmlMarkup object in the background which does the actual XML writing.

All builders will be used through the open method, which can be passed either a Builder::XmlMarkup object, or the options to create one.
